Form

class Form

Get and set the HTML form field values.

Properties

Property Description Type Default
formName the form name used as value in the data-rg-form attribute string
debugOptions {setControl:boolean, setControls:boolean, getControl:boolean, getControls:boolean, delControl:boolean, delControls:boolean} object

Methods

Use this methods in the controller after "Cookie" class is instantiated.

  • setControl (key :string, val :string) :void

    Set the form control value.
    key - the value of the "name" HTML attribute
    val - the value

  • setControls (obj :object) :void

    Set multiple form controls with one object.
    obj - the object which represent the object values, or example: {name:'John Doe', age:23, employed:true}
    NOTICE: If the obj has subproperty it can be used in the name, for example: const fruit = {seller: {name: 'Drog Ltd'}; this.testForm.setControls(fruit); then in the view can be <input type="text" name="seller.name">

  • getControl (key :string, convertType: boolean) :string|number

    Get the form control value. Returned value is array for checkboxes and select-multiple.
    key - the value of the "name" HTML attribute
    convertType - to convert the type, for example '5' to Number or '{"a": 22}' to Object. Default is true.

  • getControls (keys :string[], convertType: boolean) :object

    Get the form control values. Returned value is the corresponding object.
    keys - the values of the "name" HTML attributes (This is an JS Array!)
    convertType - to convert the type, for example '5' to Number or '{"a": 22}' to Object. Default is true.

  • delControl (key :string) :void

    Empty the form control value.
    key - the value of the "name" HTML attribute

  • delControls (keys :string[]) :void

    Empty the form control values.
    keys - the values of the "name" HTML attributes (This is an JS Array!)

Example


const { Controller, syslib } = require('regoch-web');

class MyCtrl extends Controller {
  constructor(app) {
    super();
  }

  async init () {
    this.myF = new syslib.Form('myForm');
  }

  test() {
    const firstName = this.myF.getControl('first_name');
    console.log(firstName)
  }

}
  

<form data-rg-form="myForm">
  <input type="text" name="first_name">
</form>
