eventEmitter

instance of the class EventEmitter

Emit or listen the events on an easy way simmilar to NodeJS environment. All events are attached to "window" object.

Methods

Use this methods in the controller after "Cookie" class is instantiated.

  • emit (eventName :string, detail :any): <void>

    Create and emit the event with the custom detail data.

  • on (eventName :string, listener :Function): <void>

    Attach the event listener. The listener function will be executed on every catched event.

  • once (eventName :string, listener :Function): <void>

    Attach the event listener. The listener function will be executed only first time on catched event.

  • off (eventName :string, listener :Function): <void>

    Remove the event listener for specific event.

  • offAll (eventName :string): <void>

    Remove all event listeners for specific event.

  • getListeners (): <object[]>

    returns all the active event listeners.

Example


const { Controller, syslib } = require('regoch-web');

class MyCtrl extends Controller {
  constructor(app) {
   super();
  }

  async init(trx) {
     this.eventEmitter = new syslib.eventEmitter;
  }

  test() {
    this.eventEmitter.emit('myEvt', {a: 33});
    this.eventEmitter.on('myEvt', event => { console.log(event.a); });
    this.eventEmitter.off('myEvt');
  }

}
