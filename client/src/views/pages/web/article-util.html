util

instance of the class Util

A library of utilities and helpers.

Methods

Use this methods in the controller.

  • async sleep (ms:number): Promise<void>

    make some time delay in miliseconds

Example


const { Controller, syslib } = require('regoch-web');

class MyCtrl extends Controller {
  async test() {
    await syslib.util.sleep(1300); // wait for 1.3 seconds before sending HTTP request
    const answer = await this.httpClient.askJSON('https://jsonplaceholder.typicode.com/todos/1');
  }
}
