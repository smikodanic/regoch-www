navig

instance of the class Navig

A library for navigate through the URLs by changing browser's address bar data.

Properties

Property Description Type Default
previous prvious uri and controller {uri:string, ctrl:Controller} object
current current uri and controller {uri:string, ctrl:Controller} object

Methods

Use this methods in the controller.

    SETTERS & GETTERS

  • setCurrent (ctrl :Controller, trx :object) :void

    Set current uri and controller instance {uri:string, ctrl:Controller}.
    ctrl - instance of the current controller
    trx - regoch-router transitional variable for matched route {uri:string, body:any, params:object, query:object, routeParsed:object, uriParsed:object}

  • setPrevious () :void

    Set previous uri and controller by copying the current uri and controller.

  • getCurrentURI () :string

    Get current browser bar uri which includes path and query string without hash, for example /product/13?limit=25
    (window.location.pathname + window.location.search).



    • NAVIGATION

  • goto (url :string, state :string, title :string) :void

    Navigates to a view using an absolute URL path. The controller will be executed. More at https://developer.mozilla.org/en-US/docs/Web/API/History/pushState
    url - absolute URL path, for example: '/customer/product/25'
    state - The state data. Fetch it with event.detail.
    title - Not used in major browsers.

  • goblind (url :string, state :string, title :string) :void

    This method will only change the URL in browser address bar without executing the controller.
    url - absolute URL path, for example: '/customer/product/25'
    state - The state data. Fetch it with event.detail.
    title - Not used in major browsers.

  • forward () :void

    Go forward like forward button is clicked.

  • back () :void

    Go back like back button is clicked.

  • go (delta :number) :void

    Loads a specific page from the session history. You can use it to move forwards and backwards through the history depending on the delta value.
    delta - history index number, for example: -1 is like back(), and 1 is like forward()

  • reload () :void

    Reloads the page like refresh button is clicked.



    • LISTENERS

  • onPushstate (listener :Function) :void

    Listen for the 'pushstate' event, for example when data-rg-href is clicked or when goto() is used.
    listener - callback function which will be executed on 'pushstate' event ,for example pevent => { ... } where pevent is pushstate or popstate event

  • onPopstate (listener :Function) :void

    Listen for the 'popstate' event, for example when BACK,FORWARD button is clicked or when back(),forward() is used.
    listener - callback function which will be executed on 'popstate' event ,for example pevent => { ... } where pevent is pushstate or popstate event

  • onUrlChange (listener :Function) :void

    Listen for the URL change i.e. 'popstate' or 'pushstate' event. the URL change include any URL change even hash part modification.
    listener - callback function which will be executed on 'popstate' or 'pushstate' event ,for example pevent => { ... } where pevent is pushstate or popstate event

  • onHashChange (listener :Function) :void

    Listen for the URL hash part change.
    listener - callback function which will be executed on 'hashchange' event ,for example pevent => { ... } where pevent is pushstate or popstate event

Example


const { Controller, syslib } = require('regoch-web');

class MyCtrl extends Controller {
  async runGOTO() {
    syslib.navig.goto('/admin/products/list');
  }
}
