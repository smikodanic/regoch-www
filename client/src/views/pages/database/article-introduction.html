Introduction

Regoch database based on the natural file system. Enormously huge storage capacity for Big Data. Simple scalability. The realtime engine build on the top of the regoch-websocket technology. Master-slave infrastructure.




Feature


- easy and simple installation
- scalabillity - easy to add/remove resources
- fast search on large data volume - search engines
- realtime database like Firebase but with much bigger volume
- websocket connections - direct frontend connections - no APIs needed
