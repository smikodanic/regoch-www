Controller Hooks Test

The example shows in which order will be executed controller constructor and the lifecycle hooks. Open the browser console to see the constructor and hook arguments.
  1. constructor(app) - Will be executed on the browser page reload. The app is the App class instance.
  2. loader(trx) - trx is the regoch router transitional variable. It contains the current URL and other injected data. (initialised by the router)
  3. init(trx) - trx is the regoch router transitional variable.
  4. rend(trx) - trx is the regoch router transitional variable.
  5. postrend(trx) - trx is the regoch router transitional variable.
  6. destroy(trx) - trx is the regoch router transitional variable.
